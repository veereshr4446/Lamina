## 🚀 About LUMINA

LUMINA is an AI-powered study assistant that helps students learn faster and smarter. Built with modern web technologies, it offers four powerful features to enhance your learning experience.

---

## ✨ Features

| Feature | What It Does |
|---------|--------------|
| 📄 **Smart Summarizer** | Paste any text and get key points instantly |
| 🃏 **AI Flashcards** | Convert notes into interactive Q&A cards |
| ❓ **Quiz Generator** | Create practice tests from any topic |
| 💡 **Explain Simply** | Complex topics explained in simple language |
| 🔊 **Text-to-Speech** | Listen to summaries and notes |
| 📋 **Copy to Clipboard** | One-click copy AI responses |
| 📥 **Download Notes** | Save responses as HTML files |
